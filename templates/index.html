What is Newton-Raphson?

A root-finding method that uses tangents to improve an initial guess.

Newton-Raphson Method

The Newton-Raphson method is an iterative numerical technique used to approximate the solution of \( f(x)=0 \). It starts with an initial guess, evaluates the function and its derivative, then repeatedly improves the estimate using tangent-line behavior.

Read short history and applications

Short History

The method is associated with Sir Isaac Newton, who developed tangent-based approximation ideas, and Joseph Raphson, who presented a clearer iterative algebraic form. Today, it is commonly called the Newton-Raphson method.

Where It Is Used

  • Engineering: solving nonlinear equations in circuits, control systems, structural analysis, and fluid dynamics.
  • Computation: used in scientific computing, optimization, simulations, and equation solvers.
  • Modeling: useful in physics, chemistry, economics, and mathematical models.

Complete Study Guide

Detailed Newton-Raphson Discussion

This discussion explains the whole idea behind Newton-Raphson root finding, from the meaning of a root to the purpose of every variable in the iteration table. Read this before using the calculator so the output makes sense instead of looking like random numbers.

1. What is root finding?

Root finding means looking for an input value that makes a function equal to zero. In mathematical form, the problem is written as:

\[f(x)=0\]

A value that satisfies this equation is called a root, zero, or solution of the equation. On a graph, a real root is usually seen where the curve crosses or touches the x-axis.

Example: If \(f(x)=x^3-x-2\), we are not only evaluating the function. We are searching for the value of \(x\) that makes \(x^3-x-2=0\).

2. Why use a numerical method?

Some equations are easy to solve exactly, such as simple linear or quadratic equations. However, many engineering and science equations are nonlinear, complicated, or impossible to solve neatly by hand. Numerical methods solve this by producing approximations that become more accurate with each iteration.

Newton-Raphson is popular because, when the starting guess is good and the derivative is not zero, it usually converges very quickly compared with slower bracketing methods.

3. The main Newton-Raphson formula

\[x_{n+1}=x_n-\frac{f(x_n)}{f'(x_n)}\]

The formula starts from the current estimate \(x_n\), then subtracts a correction term \(\frac{f(x_n)}{f'(x_n)}\). This correction is based on the tangent line at the current point. The x-intercept of that tangent line becomes the next estimate \(x_{n+1}\).

In simple terms, the method asks: “If I draw the tangent line here, where does that tangent line hit the x-axis?” That new x-value becomes the next guess.

4. Where the formula comes from

At the point \((x_n, f(x_n))\), the tangent line has slope \(f'(x_n)\). Using the point-slope form of a line:

\[y-f(x_n)=f'(x_n)(x-x_n)\]

The next approximation is where the tangent line crosses the x-axis, so set \(y=0\) and solve for \(x\):

\[0-f(x_n)=f'(x_n)(x_{n+1}-x_n)\]
\[x_{n+1}=x_n-\frac{f(x_n)}{f'(x_n)}\]

5. Variable names and their purposes

\(f(x)\)The original equation or function. The goal is to make this equal to zero.
\(f'(x)\)The derivative of the function. It tells the slope of the tangent line.
\(x_0\)The first guess or starting estimate chosen by the user.
\(x_n\)The current approximation during iteration number \(n\).
\(x_{n+1}\)The next approximation produced by the formula.
\(n\)The iteration counter, usually starting at 0.
\(\varepsilon\)The tolerance. This is the accepted error limit for stopping.
ErrorUsually computed as \(|x_{n+1}-x_n|\). Smaller error means the estimates are stabilizing.

6. Complete solving flow

  1. Rewrite the problem: make sure the equation is in the form \(f(x)=0\).
  2. Differentiate: find \(f'(x)\), because Newton-Raphson needs the slope.
  3. Choose a starting value: pick \(x_0\), preferably near the expected root.
  4. Evaluate: compute \(f(x_n)\) and \(f'(x_n)\) using the current value.
  5. Substitute: place those values into \(x_{n+1}=x_n-\frac{f(x_n)}{f'(x_n)}\).
  6. Compute the error: use \(|x_{n+1}-x_n|\).
  7. Decide: if the error is less than \(10^{-6}\), stop. If not, repeat using \(x_{n+1}\) as the new \(x_n\).

7. Why tolerance matters

The tolerance controls how close two consecutive estimates must be before the method stops. This project uses:

\[\varepsilon=10^{-6}=0.000001\]

This means the process stops only when the difference between \(x_{n+1}\) and \(x_n\) is smaller than 0.000001. A smaller tolerance gives a more precise answer, but it may require more iterations.

8. What the iteration table means

The calculator table shows how the approximation changes each round. The column \(x_n\) shows the current guess, \(f(x_n)\) shows how far the function is from zero, \(f'(x_n)\) shows the slope, \(x_{n+1}\) shows the improved guess, and the error shows how much the guess changed.

When \(f(x_n)\) becomes close to 0 and the error becomes smaller than the tolerance, the approximate root is accepted.

9. When Newton-Raphson works well

  • The function is differentiable near the root.
  • The derivative \(f'(x_n)\) is not zero or extremely small.
  • The starting guess is reasonably close to the actual root.
  • The graph is not too flat or badly shaped near the selected guess.

10. Common mistakes and warnings

  • Wrong derivative: an incorrect \(f'(x)\) can lead to a wrong answer.
  • Bad starting guess: the method may converge slowly, go to another root, or diverge.
  • Zero derivative: division by zero makes the formula impossible to use.
  • Forgetting parentheses: expressions such as \(\sin(x)\), \(\sqrt{x}\), and powers must be typed carefully.

11. Input guide for the calculator

In the calculator, type the variable selected in the variable box, such as \(x\), \(y\), or \(z\). You may use powers and common functions such as:

Examples: x^3 - x - 2, cos(x) - x, sqrt(x) - 2, exp(x) - 3, and log(x).

12. How to read the answer correctly

The final root is an approximation, not always an exact symbolic answer. For example, when the calculator says \(x \approx 1.521380\), it means substituting that value into \(f(x)\) makes the function very close to zero. The word approximate is important because numerical methods usually stop based on a tolerance instead of producing an exact fraction or radical form.

A good final answer should include the approximate root, the tolerance used, and whether the method converged successfully.

13. What convergence means

Convergence means the generated values \(x_0, x_1, x_2, \ldots\) are moving closer to a stable value. In Newton-Raphson, this usually happens when each new tangent-line estimate lands nearer to the root. If the values jump around, grow very large, repeat without improving, or reach the maximum iteration limit, the method may not have converged for that chosen starting guess.

When convergence is successful, the error column usually gets smaller each row until it becomes less than the tolerance.

14. Why the derivative is required

The derivative \(f'(x)\) represents the slope of the curve at the current estimate. Newton-Raphson uses this slope to decide the direction and size of the correction. A steep slope usually gives a controlled correction, while a slope close to zero can create a very large correction because the formula divides by \(f'(x_n)\).

\[\text{correction}=\frac{f(x_n)}{f'(x_n)}\]

This is why the calculator asks for both the function and the derivative, and why the auto-derivative feature is helpful for avoiding manual derivative mistakes.

15. Quick checklist before solving

  • Make sure the equation is written as \(f(x)=0\).
  • Choose the correct variable: \(x\), \(y\), or \(z\).
  • Check that the derivative matches the function.
  • Use a reasonable starting guess near the expected root.
  • Use \(10^{-6}\) tolerance for this PIT requirement.
  • Confirm that the final error is smaller than the tolerance.

Step-by-Step Procedure

The method repeats a tangent-based formula until the tolerance is satisfied.

1

Choose an initial guess

Select \(x_0\), preferably close to the expected root of \(f(x)=0\).

2

Evaluate values

Compute \(f(x_n)\) and \(f'(x_n)\).

3

Compute next approximation

Use:

\[x_{n+1}=x_n-\frac{f(x_n)}{f'(x_n)}\]
4

Check tolerance

Stop when \(|x_{n+1}-x_n|<\varepsilon\). This project uses \(\varepsilon=10^{-6}\).

Given:

\[f(x)=x^3-x-2\] \[f'(x)=3x^2-1\] \[x_0=1.5,\qquad \varepsilon=10^{-6}\]

\[x_{n+1}=x_n-\frac{x_n^3-x_n-2}{3x_n^2-1}\]
